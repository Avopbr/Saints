About

It costs R2 for us to host your event. This is a weekly charge i.e if your event is 2 weeks from now you will have to pay R4. The event will be romoved when the date passes.

We try and keep the charges minimal. If you have any queries, do not hesitate to email us.

Spot App is an event advertising hub. We advertise events for a fee. Educational events are advertised for free, simply email us with the following details:

  1. Event name
  2. Venue
  3. Image 300px * 250px
  4. Description of what event is about
  5. Venue coordinates i.e latitude and longitude
  6. Date
  7. Time

Events are added every 4 hours so keep checking for events happening near you!
